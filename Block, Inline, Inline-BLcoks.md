# Block-level Elements

* **Full-width** by default
* Each starts with a **new line**
* Width & Height **can** be set
*  ```<div> <p> <h1> <ul> <footer>```


# Inline Elements
* Side by side
* Takes space as much as needed
* Width & Height **can't** be set
